\doxysubsection{Detailed Description}
PWM library for AVR-\/\+GCC. 


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include\ <\mbox{\hyperlink{pwm_8h}{pwm.h}}>}\ }

\end{DoxyCode}


The library contains functions for controlling Timer\textquotesingle{}s 1 pin(s) with PWM.

\begin{DoxyNote}{Note}
Based on AVR Libc Reference Manual. Tested on ATmega328P (Arduino Uno), 16 MHz, AVR 8-\/bit 
\end{DoxyNote}
