Detailed Description

PWM library for AVR-GCC.

#include <pwm.h>
pwm.h

The library contains functions for controlling Timer's 1 pin(s) with PWM.

Note
Based on AVR Libc Reference Manual. Tested on ATmega328P (Arduino Uno), 16 MHz, AVR 8-bit
