Detailed Description

ADC library for AVR-GCC.

adc.h

The library contains functions for controlling Analog-digital converter.

Note
Based on AVR Libc Reference Manual. Tested on ATmega328P (Arduino Uno), 16 MHz, AVR 8-bit
